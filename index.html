Religion, Politics, and Baby Names

Using trends in baby names to predict religious trends

Introduction

For our project we attempted to examine the connection between trends in baby names and other social trends like religiosity and election results. Specifically, we demonstrate a relationship between the preponderance of biblical names in a state and Christianity in that state. Then, by drawing a connection between how Christian a state is and how socially conservative it is, we attempt to use trends in baby names to directly predict the outcome of presidential elections in different states.
